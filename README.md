# gllacy

Магазин мороженого.

## Инструкция по применению

### Установка модулей

```bash
npm ci
```

### Сборка проекта в папку build

```bash
npm run build
```

### Запуск в режиме разработки

```bash
npm run dev
```

### Запуск линтеров и валидаторов

```bash
npm test
```

### Запуск автоформатирования

```bash
npm run format
```
